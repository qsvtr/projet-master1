# deploy smart contract ERC721 with truffle

[fork and modified from this tutorial](https://bezkoder.com/node-js-jwt-authentication-mysql/)

## Installation
`yarn install`

## Setup
`cd app/config`

`cp auth.config.example.js auth.config.js` 

`cp db.config.example.js db.config.js`

Then modify this two files with your credentials

## Start
`yarn start`
